# Missions (concept)

Voxera treats every request as a Mission Card:

Observe -> Suggest -> Simulate -> Approve -> Apply -> Verify -> Remember

This scaffold includes:
- skill runner
- policy + approvals
- audit trail

This scaffold now includes built-in mission templates (`work_mode`, `system_check`) and CLI mission execution.

Next step: wire intent planning to select and parameterize missions via brain adapters.
